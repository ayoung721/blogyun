AI Agent를 활용한 블로그 제작 과정

🤖 AI Agent를 활용한 블로그 제작 과정

1. 사용한 AI 도구

이번 블로그 제작 과정에서는 AI Agent를 적극적으로 활용하였다.
주로 ChatGPT(OpenAI)를 사용하여 코드 작성과 아이디어 구성을 진행하였다.

AI를 활용함으로써 기존보다 훨씬 빠르게 웹 페이지를 제작할 수 있었다.

2. AI 활용 내용

AI는 다음과 같은 작업에 활용되었다.

  • HTML 구조 생성
  • CSS 스타일 디자인 (다크모드 UI)
  • 블로그 콘텐츠 작성 보조
  • 오류 수정 및 코드 개선

특히 디자인과 레이아웃 구성에서 많은 도움을 받을 수 있었다.

3. 실제 프롬프트 예시

아래는 실제로 사용한 프롬프트의 일부이다.

  • "블로그 HTML 구조를 만들어줘"
  • "다크모드 스타일의 카드 UI를 CSS로 만들어줘"
  • "블로그 글 형식의 예시를 작성해줘"

프롬프트를 점점 구체적으로 수정하면서 원하는 결과를 얻을 수 있었다.

4. 결과 및 느낀 점

AI를 활용한 개발은 매우 효율적이었다.
초기 구조를 빠르게 만들 수 있었고, 아이디어를 얻는 데도 큰 도움이 되었다.

하지만 AI가 생성한 결과를 그대로 사용하는 것이 아니라, 직접 수정하고 이해하는 과정이 중요하다는 것을 느꼈다.

이번 과제를 통해 AI는 개발을 도와주는 강력한 도구라는 것을 체감할 수 있었다.

앞으로도 다양한 프로젝트에서 AI를 적극적으로 활용해보고 싶다.
